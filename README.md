# Hospital Patient Management System

Python | Data Structures | Console Application

## Project Overview

This project is a console-based Hospital Patient Management System developed using Python. 
The system allows hospital staff to manage patient records, prioritize emergency cases, 
and maintain patient queues efficiently.

The program demonstrates the use of data structures, file handling, and object-oriented 
programming concepts.

## Features

- Add new patient records
- Search patient by ID
- Delete patient records
- Display all patient details
- Emergency patient priority handling
- Normal patient queue management
- Serve patients based on queue type
- Save and load patient records using CSV files
- Display patient statistics

## Data Structures Used

- Dictionary → Fast patient lookup
- Priority Queue (Heap) → Emergency patients
- Queue (Deque) → Normal patient queue

## Technologies Used

- Python
- CSV File Handling
- Object-Oriented Programming
- Standard Python Libraries
